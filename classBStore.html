Detailed Description

This class Is a dynamic data storeage class and can be used to store variables of any type listed by ASCII key. The storage of the varaible is a binarystream.

Templated setter function to assign vairables in the BoostStore from pointers.

Parameters
nameThe key to be used to store and reference the variable in the BoostStore.
inA pointer to the object to be stored. The function will store the value the pointer poitns to in the archive and keep hold of the pointer to return to Get calls later.
persistIndicates if the object that is being pointed to is stored or not. If not only the pointer is retained for later use but not in the archive and so will not be saved with the BoostStore on save calls.
